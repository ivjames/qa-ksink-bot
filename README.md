# QA KSink Bot

Visible-browser QA automation runner for the QA KSink demo site.

The target site lives in `ivjames/qa-ksink-site`.

## Local setup

Create a Python virtual environment, install `requirements.txt`, then install the Playwright browser package.

## Run locally

Set `BASE_URL` for the frontend and `API_URL` for the backend API, then run pytest in headed mode.

## Demo runner

The `run_demo.py` script is intended to launch tests in a visible browser for live demonstration.
